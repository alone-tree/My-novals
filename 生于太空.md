# 楔子

一辆运输机到达了临界发射高度。伴随着一阵轻微的振动，运输机背上的小飞机启动推进引擎，在稀薄的大气层中继续上升，而承载它的运输机则开始降高度，准备回到地面，准备下一趟运输。叫做小飞机并不十分合适，更专业的名称是航天飞机。不过相比于下方的运输机，航天飞机的大小着实显得很可爱。在航天飞机上，张辰星还在艰难地忍受着加速带来的超重感。

终于，在漫长的加速之后，航天飞机发动机熄火，进入漂移阶段。张辰星久违的失重感终于回来了。

“回家真好”，他一声轻叹。

# 楔子

2077年，人类终于建立了第一座商用的聚变发电站。尽管可以从地球的汪洋大海中提取重氢用作聚变燃料，但在实践中，这一方法并没有如前人预料的那样一劳永逸地解决人类的能源问题。原因之一在于，海水提氢需要对海水进行浓缩，而大量浓缩后的海水带来的污染仍然难以解决，大量的海水加工也对沿海地区的生态造成了严重影响。同时，随着发电技术增长，电力需求也随之增加，海水所能提供的能量逐渐力不从心，因此对巨变燃料的需求十分强烈。

而针对月球的勘探结果表明，月球含有丰富的氦元素。用氦发电不但安全高效，现有的技术也完全支持将燃料替换为氦。既然月球表面含有这一宝藏并且非常易于开采，剩下的问题就是如何挖出来并运回地球。显然，在地表建造工场再送到月球是行不通的。不过可以把大型设备的零件都运上去，在太空和月球进行组装。

由于地心引力和大气层的双重阻碍，任何将大型机械发射升空的行为都需要无比高昂的成本。好在当时已经有了成熟的两段式发射技术，第一段在大气中利用飞机的升力将火箭运送到高空，到达极限高度，空气稀薄之后，再由传统的工质引擎推动航天飞机进入太空轨道。这一技术的好处在于，发射时的加速度过载相比火箭而言小的多，因此对人体的要求要小的多，可以让人大量地进入太空，并且每单位质量的运输成本也比较低。靠着多个大国资本的投入和蚂蚁搬家式的运输方式，多年以后人类终于在太空搭建起了第一座工厂，称为母厂。之后，母厂又造出了另一座新的工厂，然后又一座......

当然，这几座工厂的使命主要是测试相关的技术，之后的计划是将母厂运到月球附近，拆卸，降落，最后在地表重新组装。当然，这是以后的事情了。在搭建完5座工厂后，这些厂由一些大型冶炼集团运作，用于生产太空金属制品。这么做的主要原因，在于政府钱不够多，因此需要吸引商业资本，当时称为PPP模式。同时，一个私人运转的工厂，也为政府省去了养护开销。

我就出生在其中的一座工厂。

# 第一章 意外的出世

我叫张辰星，名字是星辰的意思。我的父母都是冶炼厂的工人，不过不同于大多数厂工的地方在于，这座钢厂的生产要求失重环境，因此位于太空轨道。虽然当时工厂的大部分流程已经实现了自动化生产，但硕大的一座工厂，总得有人值守，不时也需要进行检修。

驻厂需要3个人，每两个月轮班一次，主要工作是在上面呆着，每天巡查一趟，防止潜在的风险。在一个四周都是钢铁、电灯，窗外则是深不见底的宇宙深空中，要度过两个月的时间，对人的耐心相当有考验。尽管可以通过网络与地面的世界联络，但是天上嘛，条件有限，网并不总是很好。事实上，厂里的网络总是不好。因此驻厂的大部分时间里，工作都很无聊。唯一的好处是，钱给的比较多。

在常规巡查之外，每隔几年还需要进行一次全面检修。一次全面检修需要大约一周时间，地面派上来的工人和驻厂工人需要在这一周时间内把工厂每一颗螺丝钉和检查螺丝钉的检测器都检查一遍。当然，主要是关注检测器。故事就发生在某一次全面检修期间。

在漆黑的太空和狭小的工厂，大多会感到很无聊。恰好这时候地面来了一批活人做检修，就很容易在这几周的工作中产生感情交流。虽然工厂明确规定，在太空不允许有任何两性的性行为，但规定往往是用来打破的。而我就是破坏规则的产物。

这一规定并不是故意为难人。尽管人类已经进入太空工业时代，大规模太空运输也已经稀松平常，但在太空待久了，仍然会不可避免的受到失重的影响，肌肉无力、骨质疏松等，导致回到地面上之后需要一段时间进行恢复。并且在起飞和返回时的过载仍然很明显，而这种过载对于孕妇和胎儿都十分危险。过去的动物实验表明，过载会显著增加流产的概率。同时，太空中的失重和辐射对胚胎发育的影响也很复杂，因此为了防止潜在的医学和人道主义灾难，起飞之前，所有的人都需要服用短效避孕药，并且女性还需要进行受孕检查，以免发生意外。

所以我的出现实际上算是一场事故，在当时甚至引发了一场世界范围内的轰动。

讨论的内容很多，有讨论是否要把我接回地面的声音，但这一提案最终被否定了，因为怕我娇嫩的身躯承受不住飞行过载和地面的重力。有关心我以后的生活，给我捐款的。也有声讨我父母的，说他们不负责任，让我承担这样的痛苦云云......

而在这些声音之外，许多科学家也注意到了我。因为任何让人在太空怀孕、出生的实验都无法通过伦理审查，科学家们对此一直都只能通过动物实验来进行推测，而直接的证据，是没有的。因此他们一直很关心我，或者说关注我。

不管怎样，我最终留在了这座工厂。而关于我的新闻，也很快被其他的新闻所淹没。

# 第二章 幼儿园

小时候我主要是靠保育员的李叔叔他们带大的。听他们说我小时候很难带，经常哭闹，上厕所喂奶也很折磨人。不过我一点印象也没有，所以我不认（笑）。

小时候我很好奇，整个工厂到处飘。除了冶炼仓和聚变仓的内部，剩下剩下的各个地方我都逛遍了。而且因为是厂里为数不多的大活人，所以大家都很照顾我，在那些哥哥姐姐的无聊时光里，我就成为了他们最好的玩具（大误）。不过我大部分的时间还是待在生活舱里，准确的说，是生活舱的重力室里面。

平时主要是驻厂的哥哥姐姐们负责照顾我，以姐姐们为主，因为哥哥们不太耐得住寂寞，不愿意干这个活，因此上来的大多数是姐姐。除了工厂的工人之外，保育员的Salim园长和李老师也时不时会上来，不过得等他们在地面放假的时候。至于父母，因为在工作中出现了严重违反规定的行为，早就被开除了，也没有来看我。

在上学以前，我的生活只有三件事：体检、锻炼，和到处瞎逛。虽然地方不大，但是小孩子嘛，总是闲不住，总想探索世界。只要进的去，整个厂可以说没有我没去过的地方。

除了做各种检查测试，我还要在重力仓做很多训练。重力仓可以说是全世界最无聊的地方，一个巨大的桶一直不停地转，桶壁上模拟地面的重力。这本来是睡觉的地方，所以当大家都去工作的时候，我就只能一个人呆在这里，还不准我爬到中间，只能在边缘活动，后来因为需要上学，又装了一个全息投影室，才稍微变得有趣一点。所谓的锻炼也不过是走来走去，跑跑跳跳之类的。我还是比较喜欢在无重力的地方，自由的飞来飞去。少部分时候，我可以跟着她们一起去上班，然后在旁边看牵引车把一大块铁扔进锅炉，出来就变成了各种形状的奇怪东西。


在所有驻厂的人里面，我最喜欢的是Jani姐姐，她和我母亲之前是好朋友。书上说母亲的姐妹应该叫做阿姨，但是“阿姨们”都觉得这样叫着显老，所以我都统一叫姐姐了。喜欢Jani姐姐的原因在于，她是我最常见的人，因为她几乎每隔一两次就会上来，会给我带一些玩具，还会给我讲好多的故事。听Jani说，她小时候的梦想就是当保育老师，只不过没选上，就只好算了。保育员的几位老师也会给我讲故事，但隔着屏幕的感觉总是没有面对面好。而且，苏老师虽然是我的“监护人”，但毕竟离得太远，还有驻厂的工人陪我，所以他们也只能选择放养我。

# 第三章 小学生活

开始上小学以后，生活就发生变化了。我需要按时上课，还要完成课后作业，李老师他们还会监督我写作业。虽然作业不多，上课的老师也很好，不过我还是很讨厌。最主要的原因大概是，重力。虽然待久了以后慢慢就习惯了，但每次回到屋里都需要做好久的心理建设。不过上学也有好处，因为我能有机会和地面的老师同学们“共处一室”，虽然也只能通过投影室看大家，没有机会和他们握手打闹，不过我也很满意了。为了支持这套投影设备，厂里专门加强了工厂和地面的网络，实现了无延迟通讯。听Jani说，要不是学校的校长给厂长施压，她们甚至没办法好好在天上打游戏。

上学嘛，总是很快活的。毕竟可以认识好多新同学。同学们都很好奇我的上课环境。虽然现在的太空工业已经比较成熟了，太空旅游也相当平民化，但大部分游客的体验仅限于感受失重环境和浩瀚的星空，时间不长，空间不大，并且除了星星也没什么好看的。

于是乎，我的课余活动就多了一项：带同学参观太空工厂，顺便表演一下太空飘移技术。当然，离开我的那个房间之后，就只能戴头盔了。他们也经常带我参观地面，总的来看还是他们带我玩的比较多。不管是去参观动植物园、去郊外春游，每次出去活动老师都会特别照顾我一下。虽然只能眼睛看看，聊聊天。我有时候也想去地面上和他们一起玩耍，但是苏医生都不准，说我的身体条件不允许，让我多锻炼。

带我玩的人主要是小圆。小圆叫做罗亮，因为人长的比较圆润而得其称。和我熟了以后，他开始叫我大圆，因为我比他更圆润一点点。不过我长的圆润可能是受太空环境影响，他大概得怪嘴馋。除了吃，小圆的另一大爱好是逛公园，各种园林，动植物园，自然保护区等等。显然，这不是经常能找到同伴的爱好，不过我因为整天待在这个工厂内，对这些地方倒是无比向往，所以每次他出去玩我都让他带着我。说是带着我，其实也不过是带上我的眼睛。

小圆的父母人很好，为了方便我们俩交流，专门给他买了全息头盔。头盔四周布有多个摄像头和麦克风，能拍摄周围的全景范围，通过另一个人的眼镜呈现，而且视角相对自由，不会受到对方转头的影响，就好像站在那个人身边一样。这种设备最早用于在地面远程指导太空工厂建设。为了避免长期的失重对身体造成影响，太空工人一般一、两个月就需要返回地面修整。工人可以多招募一些进行轮岗，但一些技术要求很高，难以替代的岗位就比较麻烦，施工的时候又不能没有这些人在场，因此为了让这些人能够长期亲临现场工作，就发明了这样的替代工具，让这部分人在地面也能像在现场一样看到周围的环境。后来这种设备逐渐被分居两地的家庭采用，体积也逐渐轻量化，最后就变成了头盔的样子。如果接收端的条件允许的话，可以把周围通过专门的球形投影室呈现，就比如我的房间那样，能实现身临其境的效果。

于是大家出去玩的时候，小圆就会带上我的眼睛，找个支架把头盔架起来，我就能在一边看其他人。时不时的，小圆的父母会带他去城市外面，这个时候，他们家一般会带上我。他喜欢看看鸟，我喜欢看树，一路上倒也挺聊的来。每次去玩回来，我都觉得自己身处的这一方空间是多么逼仄狭窄，我渴望回到地面上。但是每次身体评估的结果都不是很好，一个是当时年纪尚小，身体还没长好，加上我经常会去工厂里的失重空间闲逛，所以骨骼和肌肉都很弱。他们说要等我小学毕业才行。

小学那几年其实蛮无聊的。虽然听起来好像挺有意思，但一直以来我都很想回到地面上，越长大越想。好不容易盼到了小学毕业，但医生还是不同意，说我还在长身体，让我再等等看。于是我又在上面读完了初一。终于在临放暑假的时候，苏医生和我说可以出发了！我还和同学约好了开学见。现在想想还真是有点天真。

# 准备回家

把人从太空送回地面这个事情本身很简单，问题在于我比较弱，骨密度比常人低很多，所以得考虑怎么在这过程中不会骨折。还需要小心不被各种细菌病毒感染。最后的解决方案是，做一个可以正好卡住我的软床垫，放上各种监测仪，再做一个能装下床垫的无菌仓，最后改造一辆运输机，把无菌仓装进去，等我落地之后再转入无菌室，逐渐接触外界细菌，同时适应地面的环境。听起来事情很多，但我需要做的就是每天好好锻炼，尽量减少失重的时间，其他的都不用我操心。

出发时间定在了格林尼治标准时间2122年7月17日，为此我已经提前几天就一直待在重力仓的投影室里面，反复观看整个过程的演示动画，一方面是为了熟悉整个过程（虽然整个过程其实没我太多事情）另一方面也是给自己找点事情，好在重力仓呆着不觉得无聊。

终于，时间到了，我透过窗户看到运输机逐渐靠拢，对接，我甚至隐约感受到了对接口锁死带来的振动，虽然我知道这完全是心理作用，但还是连带着心情也开始激动起来。

苏医生来最后和我交代了一遍过程，然后开始领着我走向出口。其实不需要他带路，这条路我早就走过无数次了，每次换班的时候我都会跟着一起走到出口，甚至如果司机心情好，不赶时间的时候我还会上运输机里坐坐，幻想着什么时候能有机会跟着下去。

当天值班的几位哥哥姐姐都来送我，不过我的心思早就跑到地面去了，完全记不得他们当时说的什么。到出仓口准备进去的时候，心里突然觉得五味杂陈。穿过这个不大不小的圆圈，过去的生活将和我永远说再见，而我当时还不知道未来会是怎样。我还记得当时回头最后看了一眼，那个略显灰暗的厂房和对面明亮洁白的运输机之间只隔了一个门。而这个门现在向我敞开着。

进入运输机之后，我看到了一些熟悉的面孔，是苏医生的几位同事。之前大多时候只远程连线的时候见过，每个人都负责不同的领域。还有几个不认识，是航天局和炼钢厂的人。现在想想，我那会也算是一个高光时刻了，这么多人来接我，还准备了专门的装备。幸亏花的是他们的研究经费。

剩下的事情就很简单了，我换好了防护服，钻进了无菌仓，躺在那个量身定制的凹槽里面，和外面的人有一搭没一搭的聊天。大家都有些紧张，所以一路上没什么话。工质引擎发动的时候带来了一丝振动，提醒我现在已经出发了，除此之外没什么变化。机舱的光明亮而稳定，我以为这种平静会一直持续到我落地，然后我做一些检查，就可以开始新的生活了。但实际情况总是会比较复杂一些，进入大气层之后，我明显感受到了一阵压力，而且这个压力还一直在增大。每隔几分钟我都会问现在的重力是多少，每次他们都只我说0.5，0.8，1.1，1.2之类的，但我觉得他们应该是在骗我，让我不要想太多。因为我明明觉得已经很难受了，甚至心里在想现在是不是已经有10g的重力。虽然我也不知道10g的重力到底是什么感觉。我最后的意识停留在“即将展开降落伞，请准备”。等我再醒过来的时候，运输机就已经进入了平稳飞行的状态，和普通的民航飞机没有任何区别。

降落的过程很顺利，虽然在减速过程中晕了一次，但总归是醒过来了。又一次抖动之后，他们告诉我现在已经落地了。我第一次和传说中的“大地母亲”有了直接的接触。我很想立刻起来看看，但碍于被关在小小的无菌仓里面，只好作罢。很快，我就被连人带仓一起拖进了一辆车。坐车真是一件痛苦的事情，尤其是上坡下坡的时候。从机舱进入到地面的时候我差点吐自己一身，甚至一众的医生也很担心我再晕过去。不过还好，顺利进入了医院之后，我基本上完成了回家的第一步。现在要做的就是逐渐适应地面的重力，再慢慢接触各种微生物，以及，学会走路。

# 整体介绍

在一座太空工厂中，主角意外出生。在最初的一些岁月中依靠工厂中的重力仓勉强活了下去，但大部分时间仍然处于无重力环境下，因此对于重力的适应能力很差。在青春期即将结束的时候，身体发育大致成型，因此地面医护人员安排行程将他接回去。回到地面后因为重力和免疫问题差点死去，但最后都救回来了，成为了一个普通人。
